# Stakeholder & Public Views

*Public or semi-public access.*


## 1. System Status Page


* **Target**: Public, Customers, Support


* **Route**: `/status`


* **Goal**: Transparency during outages.


* **URL**: `status.penguinmails.com`


### Widgets


* **Component Status**: API, Dashboard, Sending Nodes, Webhooks.


* **Incident History**: Log of past incidents and resolutions.


* **Uptime Badges**: "99.99% Uptime last 30 days".


## 2. Investor Briefing Room (Secure)


* **Target**: Investors, Board Members


* **Route**: `/dashboard/analytics/investor` (Proposed)


* **Goal**: Due diligence and quarterly reporting.


### Widgets


* **Cohort Analysis Heatmap**: Retention over time.


* **Unit Economics Card**: LTV, CAC, Payback Period.


* **Market Penetration Map**: Geographic user distribution.
