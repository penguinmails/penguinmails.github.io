# Campaign Management MVP

## Overview

**Priority**: P0 (MVP blocker)
**Effort**: Large (4 weeks)
**Timeline**: Q1 2026

**Description**:
Basic campaign creation, editing, and sending flow.

**Acceptance Criteria**:

- 5-step campaign wizard (Settings, Recipients, Content, Preview, Send)
- Campaign list view (Draft, Scheduled, Sending, Sent)
- Duplicate campaign
- Send test email
- Basic personalization (Merge tags)
- Unsubscribe link handling

**Dependencies**:

- Template system
- Contact segmentation
- Queue system
