# Basic Campaign Analytics

## Overview

**Priority**: P0 (Launch Blocker)
**Timeline**: Q1 2026 (5-7 days)
**Effort**: Small

**Description**:
Real-time visibility into campaign performance, including sent, delivered, opened, clicked, bounced, and unsubscribed metrics.

## Capabilities

- Real-time metrics
- Calculated rates
- Campaign-level aggregation
- Step-level metrics
- Time-series charts
- Geographic distribution
- Device breakdown

## Dependencies

- Email pipeline with tracking
- Analytics database schema
- Campaign management system
