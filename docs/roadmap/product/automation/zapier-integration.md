# Zapier Integration

## Overview

**Priority**: P3 (Post-MVP)
**Timeline**: Q2 2027
**Effort**: Medium

**Description**:
Enable workflows to trigger external apps via Zapier and receive Zapier triggers.

## Acceptance Criteria

- Zapier webhook triggers
- Zapier action endpoints
- API key authentication
- Event payload standardization
- Zapier app listing

## Dependencies

- Basic workflow system
- Stable API endpoints
- API key management
