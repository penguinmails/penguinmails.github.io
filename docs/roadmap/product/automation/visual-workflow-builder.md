# Visual Workflow Builder UI

## Overview

**Priority**: P1 (MVP)
**Timeline**: Q3 2026 (4-6 weeks)
**Effort**: Large

**Description**:
Drag-and-drop interface for creating automation workflows with triggers, conditions, and actions.

## Acceptance Criteria

- Visual canvas with drag-and-drop
- Trigger/Action configuration
- Condition builder
- Workflow connections
- Testing/preview mode
- Save and activate functionality
- Workflow list view

## Dependencies

- Workflow execution engine
- Trigger library
- Action library
