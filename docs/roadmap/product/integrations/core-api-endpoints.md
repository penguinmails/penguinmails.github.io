# Core REST API Endpoints

## Overview

**Priority**: P0 (Critical)
**Timeline**: Q1 2026 (2-3 weeks)
**Effort**: Large

**Description**:
Implement core REST API endpoints for contacts, campaigns, emails, and analytics with standard conventions.

## Capabilities

- Contacts API (CRUD)
- Campaigns API (CRUD + actions)
- Emails API (Send + List)
- Analytics API
- Pagination, Filtering, Sorting
- Error handling

## Dependencies

- API Key Management System
- Database schemas
