# Queue & Background Jobs Roadmap

This roadmap outlines the development timeline for queue system enhancements, from current active state through future advanced capabilities.

**Future Enhancements:** As needed

* **[Email Pipeline](email-pipeline.md)** (Active): High-throughput processing (10k+ emails/min).
* **[Background Jobs](background-jobs.md)** (Active): Campaign processing, imports, and maintenance.
