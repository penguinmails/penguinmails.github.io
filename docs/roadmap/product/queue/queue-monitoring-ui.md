# Queue Monitoring UI

## Overview

**Priority**: P1 (MVP)
**Effort**: Small (1 week)
**Timeline**: Q1 2026

**Description**:
Dashboard for monitoring queue health, throughput, and error rates.

**Acceptance Criteria**:

- Real-time job counts (waiting, active, completed, failed)
- Throughput metrics (jobs/second)
- Error rate visualization
- Worker status (online/offline)
- Basic job inspection (view payload of failed jobs)
- Retry button for failed jobs

**Dependencies**:

- Queue System MVP
- Admin dashboard
