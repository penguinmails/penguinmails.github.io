# Social Login (OAuth)

## Overview

**Priority**: P2 (Post-MVP)
**Effort**: Medium (2 weeks)
**Timeline**: Q2 2026

**Description**:
Allow users to sign up and login using their existing Google or GitHub accounts.

**Features**:

- **Google OAuth**: "Sign in with Google" button.
- **GitHub OAuth**: "Sign in with GitHub" button.
- **Account Linking**: Link social account to existing email account.
- **Profile Sync**: Import name and avatar from social provider.

**Dependencies**:

- Auth service
- OAuth provider apps (Google Cloud Console, GitHub Developer Settings)
