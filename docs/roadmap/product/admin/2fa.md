# Two-Factor Authentication (2FA)

## Overview

**Priority**: P3 (Future)
**Timeline**: Q1 2026+ (Conditional)
**Effort**: Medium

**Description**:
TOTP-based 2FA with authenticator apps. Conditional based on customer demand.

## Features

- QR code setup
- Backup codes
- Trusted devices
- Tenant-level enforcement
- Admin override
- Recovery flow

- NileDB 2FA assessment
