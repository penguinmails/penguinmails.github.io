# Enterprise & Admin Roadmap

## Overview

This roadmap outlines the development timeline for enterprise features, platform administration, and security enhancements.

**Current Status:** Early MVP (36% complete)

## MVP Roadmap Items (Q1 2026)

### Critical Blockers (P0)

* **[Platform Infrastructure Monitoring Dashboard](infrastructure-monitoring.md)** (P0): Comprehensive dashboard for platform administrators.
* **[Payment & Billing Overview Dashboard](billing-overview-dashboard.md)** (P0): Admin dashboard for subscription monitoring.
* **[Account Security Enhancements](account-security.md)** (P0): Essential security features (lockout, session management).

### High Priority (P1)

* **[Workspace Management Documentation](workspace-management-docs.md)** (P1): Documentation for workspace features and permissions.
* **[Onboarding Experience Implementation](onboarding-experience.md)** (P1): Guided setup wizard and interactive onboarding.
* **[Admin Platform Features](admin-platform-features.md)** (P1): Tools for user and tenant management.
* **[Session Management UI](session-management-ui.md)** (P1): UI for viewing and managing active sessions.
* **[Staff Authentication System Documentation](staff-auth-docs.md)** (P1): Documentation for staff authentication and permissions.

## Post-MVP Roadmap (Q2 2026 - Q1 2027)

### Q2 2026: Authentication & Social

* **[Social Login (OAuth)](social-login.md)** (P3): Sign in with Google and other providers.
* **[Two-Factor Authentication (2FA)](2fa.md)** (P3): TOTP-based 2FA (Conditional).

### Q3 2026: Enterprise SSO

* **[Single Sign-On (SSO) for Enterprise](sso-enterprise.md)** (P3): SAML 2.0 SSO for enterprise customers (Conditional).

### Q4 2026: Advanced Security

* **[Advanced RBAC (Custom Roles)](advanced-rbac.md)** (P2): Custom role creation with granular permissions.
* **[Biometric Authentication](biometric-auth.md)** (P3): Fingerprint/Face ID login.
* **[Advanced Session Management](advanced-session-management.md)** (P2): Enhanced session control and security.

### Q1 2027: Compliance & Operations

* **[Audit Logs & Compliance](audit-logs-compliance.md)** (P2): Comprehensive activity logging and compliance tools.
* **[Real-Time Monitoring & SLA Tracking](sla-tracking.md)** (P2): System health monitoring and SLA guarantees.
* **[Team Collaboration Features](team-collaboration.md)** (P2): Advanced team features for large organizations.
* **[Compliance Automation](compliance-automation.md)** (P2): Automated tools for GDPR, HIPAA, SOC 2.
* **[Dedicated Support & SLA Tiers](support-tiers.md)** (P2): Premium support tiers and ticketing system.
