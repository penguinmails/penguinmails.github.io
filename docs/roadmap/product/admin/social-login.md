# Social Login (OAuth)

## Overview

**Priority**: P3 (Post-MVP)
**Timeline**: Q2 2026
**Effort**: Low-Medium

**Description**:
Sign in with Google and other OAuth providers using NileDB's built-in OAuth support.

## Features

- Implement Google OAuth
- Add additional providers (GitHub, Microsoft)
- Account linking
- Profile data sync
- Fallback to email/password
- Token refresh handling

## Dependencies

- User authentication system (NileDB)
- NileDB OAuth SDK
