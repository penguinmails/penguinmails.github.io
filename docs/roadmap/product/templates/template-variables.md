# Template Variables (Advanced)

## Overview

**Priority**: High
**Timeline**: Q2 2026 (4-5 weeks)
**Effort**: Large

**Description**:
Sophisticated personalization logic including loops, conditionals, formatting filters, and nested variables.

- Merge tags
- Conditional blocks (if/else)
- Custom variables
- Dynamic content
- Nested variables
- Date formatting
- String manipulation
- Loops

- Handlebars template engine
- Variable validation
- Variable picker UI
