# Authentication & Security Roadmap

This roadmap covers the authentication, security, and compliance features of the platform.

## MVP (Q1 2026)

* **[Password Strength Enforcement](/docs/roadmap/product/authentication/password-strength-enforcement)**: Strong password policies.
* **[Unverified Email Restrictions](/docs/roadmap/product/authentication/unverified-email-restrictions)**: Limit access for unverified users.
* **[CAPTCHA Protection](/docs/roadmap/product/authentication/captcha-protection)**: Bot prevention for forms.
* **[Account Lockout](/docs/roadmap/product/authentication/account-lockout)**: Brute force protection.
* **[Login Activity Log](/docs/roadmap/product/authentication/login-activity-log)**: User-facing security logs.
* **[Session Management UI](/docs/roadmap/product/authentication/session-management-ui)**: Manage active sessions.
* **[Email Change Verification](/docs/roadmap/product/authentication/email-change-verification)**: Secure email update flow.
* **[Remember Me](/docs/roadmap/product/authentication/remember-me)**: Persistent sessions.

## Post-MVP (2026-2027)

* **[Two-Factor Authentication](/docs/roadmap/product/authentication/two-factor-authentication)**: TOTP-based 2FA.
* **[Social Login](/docs/roadmap/product/authentication/social-login)**: OAuth integration (Google, GitHub, Microsoft).
* **[Single Sign-On (SSO)](/docs/roadmap/product/authentication/single-sign-on)**: SAML/OIDC for enterprise.
* **[Biometric Authentication](/docs/roadmap/product/authentication/biometric-authentication)**: WebAuthn/FIDO2 support.
* **[Advanced Session Management](/docs/roadmap/product/authentication/advanced-session-management)**: Enterprise session controls.
